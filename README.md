# Sudoku Visualisation

An interactive Sudoku solver with real-time visualization to demonstrate the step-by-step solving process.

## Features
- Step-by-step Sudoku solving visualization
- Dynamic puzzle generation
- User input handling for custom puzzles
- Guided solving mechanisms for better understanding
- Responsive UI for seamless interaction

## Technologies Used
- HTML
- CSS
- JavaScript
- C++ (for the solver logic)

## Live Demo
[Sudoku Visualisation](https://kousik1314.github.io/Sudoku-Visualisation/)

## Installation & Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/Kousik1314/Sudoku-Visualisation.git
   ```
2. Open the `index.html` file in a browser.

## How It Works
- Users can enter their own Sudoku puzzle or generate a random one.
- The visualization will show the solving process step by step.
- The solver uses a backtracking algorithm for efficient solving.
